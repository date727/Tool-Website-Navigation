# 网站背景图片设置说明

要更换网站的背景图片，请按照以下步骤操作：

1. 在网站根目录下创建一个 `images` 文件夹（如果还没有的话）
2. 将你选择的背景图片放入 `images` 文件夹中
3. 建议图片格式为 jpg 或 png
4. 建议图片分辨率至少为 1920x1080 像素
5. 将图片重命名为 `background.jpg`（或 .png，取决于你的图片格式）

图片位置：
```
导航网站/
  ├── images/
  │   └── background.jpg  <- 将你的背景图片放在这里
  ├── index.html
  └── README.md
```

放置好图片后，网站会自动使用你的背景图片。
